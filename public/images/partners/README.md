# Loga partnerů

Sem vlož loga partnerů jako obrázky. Očekávané názvy (viz `src/data/site.ts`):

- `neversecond.png` — logo Neversecond
- `vamos66.png` — logo Vámoš 66

Mizuno a FyzioFit se zatím načítají z původní CDN (Framer); můžeš je sem
také stáhnout a v `src/data/site.ts` přepsat cestu na `/images/partners/...`.

Ideálně PNG s průhledným pozadím, šířka ~400–800 px. Dokud soubor chybí,
zobrazí se místo loga název partnera textem.
